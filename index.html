Funktsiyani qaytarish

JavaScript bayonotga yetganda return, funktsiya bajarilishini to'xtatadi.

Agar funktsiya bayonotdan chaqirilgan bo'lsa, JavaScript chaqiruvchi bayonotdan keyin kodni bajarish uchun "qaytadi". Funktsiyalar ko'pincha qaytish qiymatini hisoblashadi . Qaytish qiymati "qo'ng'iroq qiluvchi" ga "qaytariladi":

→

Keling bularni kod sentaksisini ham korib chiqamiz

Ikki raqamning mahsulotini hisoblang va natijani qaytaring:

// Function is called, the return value will end up in x let x = myFunction(4, 3); function myFunction(a, b) { // Function returns the product of a and b return a * b;}

Funktsiyalar yordamida siz kodni qayta ishlatishingiz mumkin

Farengeytni Selsiyga aylantiring:

function toCelsius(fahrenheit) { return (5/9) * (fahrenheit-32); } let value = toCelsius(77);

Ko'p marta ishlatilishi mumkin bo'lgan kodni yozishingiz mumkin.

Noto'g'ri parametrlarga ega funktsiyaga kirish noto'g'ri javobni qaytarishi mumkin:

function toCelsius(fahrenheit) { return (5/9) * (fahrenheit-32); } let value = toCelsius();

Turli natijalarni ishlab chiqarish uchun siz bir xil kodni turli argumentlar bilan ishlatishingiz mumkin.

Sana obyektlarini yaratish

  • new Date()

  • new Date(year,month)
  • new Date(year,month,day)
  • new Date(year,month,day,hours)
  • new Date(year,month,day,hours,minutes)
  • new Date(year,month,day,hours,minutes,seconds)
  • new Date(year,month,day,hours,minutes,seconds,ms)
  • new Date(milliseconds)
